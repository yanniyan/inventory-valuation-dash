# Inventory Valuation Dashboard

A Dash-based analytics tool for calculating and visualizing inventory valuation metrics.

## Features
- Data ingestion from CSV
- FIFO / LIFO / Weighted Average valuation logic
- Interactive charts and filters
- Clean project structure with Makefile automation

## Setup
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

## Run
python app/app.py

scripts/
